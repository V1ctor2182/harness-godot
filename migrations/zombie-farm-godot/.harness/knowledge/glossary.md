# Glossary — Zombie Farm

## Game Terms
- **Zombie types**: Shambler (slow tank), Runner (fast), Brute (high damage), Spitter (ranged)
- **Quality tiers**: Bronze (×1.0), Silver (×1.3), Gold (×1.8), Iridium (×2.5)
- **Five Elements (五行)**: Metal, Wood, Water, Fire, Earth — generating/overcoming cycle
- **Cultivation realms**: Qi Refining → Foundation Building → Golden Core
- **Dark Coins**: Primary currency
- **Spirit Stones**: Premium currency
- **Mutation catalysts**: Radioactive Sludge, Corrupted DNA, Evolution Serum
- **Soil Plot**: Where zombie seeds are planted and grown

## Engine Terms
- **Node**: Godot scene tree building block
- **Scene (.tscn)**: Godot scene file (text format, contains node tree)
- **Resource (.tres)**: Godot resource file
- **Autoload**: Global singleton registered in project.godot
- **Signal**: Godot's observer pattern for node communication
- **GDScript**: Godot's Python-like scripting language
- **GUT**: Godot Unit Testing framework (addons/gut/)
- **Headless mode**: Godot running without display (`--headless`)
- **ExtResource**: Scene file reference to an external resource (uses UID in Godot 4)
