# PRD — Product Requirements

Place your product requirement documents here as Markdown files.
The Orchestrator agent reads these when planning cycles.

One file per major feature area is recommended, for example:

- `01-core-gameplay.md` — movement, farming loop, basic mechanics
- `02-zombie-types.md` — types, stats, mutations, evolution
- `03-combat.md` — auto-battle, squads, enemies, rewards
- `04-economy.md` — resources, buildings, shops, progression
- `05-world.md` — map, dungeons, NPCs, exploration
